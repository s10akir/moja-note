# moja-note

## これなに
markdownのエディタです。授業中のノート取りを快適にするために作りました。かなりKobitoに影響されています。

現在、

* オートセーブ機能
* リアルタイムプレビュー
* 補完
* ノート管理

の機能を備えています。

## install
**※現在moja-noteはアルファ版です。インストールは推奨しません。**

### need
* nodejs
* electron

`git clone https://github.com/s10akir/moja-note`

`npm install`

`electron .`
